The folder contains two files:

1) "positive_concept_pairs.txt" that contains lines like this:

positive(A, B). 

Here, (A,B) is a related concept pair generated by GPT-4.

2) "negative_concept_pairs.txt" that contains lines like this:

negative(A, B). 

Here, (A,B) is a random concept pair that should not be related. 


These two files were used to train the informativeness classifier.
